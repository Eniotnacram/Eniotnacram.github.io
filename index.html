Custom Quest Tasks - Documentation

  • Create via Unity: Create > GameData > Tasks > Custom Quest Task
  • Supported Tasks: KillMonster, GetCurrency, SkillLevel, etc.
  • Use ScriptableObjects for logic (do not use QuestSO directly)
  • Custom editor fields require correct header matching

Overrides required in PlayerCharacterEntity: OnKillMonster, OnRewardCurrency
