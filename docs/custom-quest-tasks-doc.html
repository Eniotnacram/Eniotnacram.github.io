Overview: This addon enhances the MMORPG KIT with new quest mechanics that are customizable using ScriptableObjects.

Available Quest Tasks:

  • KillMonsterTask: Tracks all or specific monster kills.
  • GetCurrencyTask: Tracks currency gained after accepting the quest. Ignores pre-owned currency.
  • GetCharacterLevelTask: Requires reaching a specific character level.
  • GetCharacterSkillLevelTask: Requires minimum skill levels. Can be set to require all skills or any.
  • GetCharacterCompletedQuestTask: Requires completion of a specific quest.
  • GetCharacterAttributesTask: Tracks total attribute points spent. You can define which attributes are tracked.

How to Create a Custom Task:

  1. In Unity, go to Create > GameData > Tasks > Custom Quest Task
  2. Select the task type (e.g., KillMonsterTask)
  3. Configure its fields and set the description
  4. Add it to a Quest under the custom quest task list

Important: Task descriptions should not be placed in the Quest ScriptableObject. Instead, use the Custom Quest Task ScriptableObject for all logic and values.

Editor Integration:

  • Tasks display editor fields through custom editors.
  • The Header string in BaseCustomQuestTask_QuestExtensions must match the task type name exactly.

Required Overrides in PlayerCharacterEntity:

  • OnKillMonster
  • OnRewardCurrency
