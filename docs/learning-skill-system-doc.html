📘 Skill Learning System - Documentation

Overview:
This addon enhances the MMORPG KIT by allowing skills to be learned over time rather than instantly. Players trigger learning manually via the UI, and skills level up after a server-defined wait period.

🧩 Setup Instructions

1. Database Setup

If using MySQL:

  1. Open your MySQL database manager (e.g., phpMyAdmin).
  2. Import the file: mysql_learningskill.sql
  3. This will create a table named characterlearningskill required for tracking learning progress.

If using SQLite: No setup is needed. The table will be created automatically at runtime by the system.

2. Player Prefab Setup

  1. Open your Player Prefab in Unity.
  2. Add the component PlayerCharacterLearningSkillComponent.
  3. This script manages learning logic and checks timers periodically.

3. Add Skill Learning Time

  1. Go to a Skill ScriptableObject (e.g., Fireball, Heal).
  2. In the Learning Settings section, set the learningDuration in seconds.
  3. If you set learningDuration = 0, the skill will be learned instantly.

🖱️ UI Integration

4. Add Skill Learn Button Event

  1. Locate your UI prefab or scene with the Skill List UI.
  2. Find the button the player presses to learn a skill.
  3. In the button's OnClick event list:
    • Drag the GameObject containing the UICharacterSkill component into the event target.
    • Choose the method: UICharacterSkill → OnClickAddLearningSkill()
  4. This will trigger a request to the server to begin the learning process.

5. Optional UI Feedback

You can visually show remaining time and a progress bar:

  • In the UICharacterSkill component:
    • Assign a UITimer component to uiTimerLearningSkillRemainingTime.
    • Assign an Image to imageLearningSkillGauge for the fill animation.
  • These will automatically update based on skill progress.

⚙️ Behind the Scenes

  • The server checks learning progress every 5 seconds via PlayerCharacterLearningSkillComponent.
  • If the configured time has passed, the skill level is increased automatically.
  • Each skill can only be learned once at a time per character.

🧪 Testing & Troubleshooting

  • If skill doesn't level up: verify the timer, database setup, and prefab component.
  • Make sure the skill has a valid duration (non-negative).
  • Ensure the player has required skill points and currency as configured.

📎 Dependencies

  • MMORPG KIT (2D/3D/Survival)
  • Database (MySQL or SQLite)
