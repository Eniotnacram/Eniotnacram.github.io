Overview: This addon enhances the MMORPG KIT with a powerful Unity Editor tool that makes it easy to manage multilingual content. You can search, edit, add, and validate translation keys across all supported languages using an intuitive interface.

Key Features:

  • Search & Edit: Quickly find translation keys and edit their values inline for each language.
  • Add Keys: Add new translation keys across all languages at once.
  • Duplicate Key Detection: Automatically identifies keys that appear more than once in a language file.
  • Missing Key Detection: Lists keys that are missing from specific languages.
  • Lock Editing: Prevent accidental changes by locking the interface.
  • Prefab & Scene Saving: Changes are saved back to the LanguageManager prefab or active scene object.

How to Open the Tool:

  1. In Unity, go to MMORPG KIT > Language Manager Search from the top menu.
  2. The tool will open in a new Editor window.
  3. You can dock this window anywhere in your Unity layout.

Setup Instructions:

  1. Drag your LanguageManager component into the field at the top of the tool.
  2. This can be either a prefab from the Project folder or a scene object.
  3. The tool will then analyze all available languages and prepare them for editing.

Search & Editing:

  • Use the search bar to find specific keys.
  • Each search result displays the key and values for all defined languages.
  • Keys and their values can be edited directly in the interface.
  • Keys can be deleted one by one using the delete button.

Adding New Keys:

  1. Switch to the Add Keys tab in the tool.
  2. Enter the new key name and its values for each language.
  3. Click Add Keys to insert the key (existing keys are skipped).
  4. Changes are saved automatically when using a prefab.

Validation Tabs:

  • Duplicate Keys: Lists all keys that appear multiple times in a language.
  • Missing Keys: Shows which languages are missing required keys.

Save & Refresh:

  • Save Changes: Saves current edits to the prefab or marks the scene as dirty if a scene object is used.
  • Refresh: Reloads the LanguageManager and updates all cached key data.

Best Practices:

  • Use the Refresh button after editing the LanguageManager externally.
  • Use Lock mode to browse safely without making changes.
  • Back up your prefab before performing batch edits.
