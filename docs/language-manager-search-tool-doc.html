Overview: The LanguageManagerSearchTool is a Unity Editor extension designed specifically for the MMORPG KIT. It allows developers to view, search, edit, add, and validate translation keys across multiple languages directly within the Unity Editor.

How to Open:

  1. In the Unity menu bar, go to MMORPG KIT > Language Manager Search
  2. The tool will open in a new Editor window
  3. You can dock it like any other Unity Editor panel (e.g., next to the Inspector or Console)

Setup:

  1. Drag your LanguageManager component into the field at the top of the tool
  2. This can be a prefab (from the Project folder) or a scene object
  3. Once loaded, the tool will analyze all available languages and their keys

Search & Edit:

  • Enter a search term to find matching keys
  • Results will show language key, the key name, and its value
  • You can edit key names and values directly in the list
  • Select and delete keys individually

Add Keys:

  1. Click on the Add Keys tab
  2. Type a new key name
  3. Enter a value for each language
  4. Click Add Keys to insert them into all languages (skips duplicates)
  5. The tool will automatically save if working with a prefab

Validation Tabs:

  • Duplicate Keys: shows keys that exist more than once in a language
  • Missing Keys: shows which languages are missing keys found in others

Save & Refresh:

  • Save Changes: saves data to the prefab (or marks scene dirty if not prefab)
  • Refresh: reloads the LanguageManager and rebuilds all cached data (Search, Add, Missing, Duplicates)

Best Practices:

  • Use Refresh if you edit the LanguageManager manually outside the tool
  • Use the Lock button to prevent accidental editing when browsing
  • Back up your prefab before performing bulk changes
