Overview: The Animation Event Component enables animation-based gameplay triggers. This system lets you play sounds, spawn effects, or run logic directly from animation clips.

Step 1: Add the Component

  • Attach the AnimationEventComponent script to your character prefab.
  • Assign the characterâ€™s Animator to the Animator field in the component.

Step 2: Configure a Trigger

  • In the Animation Triggers list, click the + button to add a new trigger.
  • Set the Trigger Name (e.g., DEATH, STEP, ATTACK).
  • For each trigger, you can assign:
    • Audio Clip or Addressable Audio Clip
    • Audio Source
    • Game Effect Prefab or Addressable Effect
    • (Optional) Transform Container to control where the effect spawns

Step 3: Add the Animation Event

  • Open the desired animation clip in Unityâ€™s Animation window.
  • Right-click on the timeline and select Add Event.
  • Set the function name to TriggerEvent.
  • In the string field, enter the exact Trigger Name defined earlier.

Important Notes:

  • Trigger names are case-sensitive. They must match exactly.
  • You can define as many triggers as you need.
  • Use this system for events like FOOTSTEPS, ATTACKS, SHOUTS, HITS, MAGIC, etc.
  • This version does not support camera shake out of the box.
