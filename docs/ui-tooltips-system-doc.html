Overview:
This addon introduces a flexible and modular tooltip display system for MultiplayerARPG. Designed for use with UI elements such as buttons or icons, it lets you show dynamic in-game information using ScriptableObjects and popup dialogs.

Key Features:

  • Uses ScriptableObject-based Tooltip assets to store title, description, and icon.
  • Centralized registration via GameDatabase and GameInstance.
  • Drag-and-drop tooltip assignment via the Unity Inspector.
  • Fully customizable popup dialog UI via UITooltipDialog.
  • Simple click-to-show behavior using the UITooltipTrigger component.

How to Set Up:

  1. Create a new Tooltip asset: Create > GameData > Tooltip.
  2. Fill in the tooltipâ€™s title, description, and optional icon.
  3. Attach UITooltipTrigger to a UI Button or Image.
  4. Drag your Tooltip asset to the tooltip field in the component.
  5. Ensure your scene contains a BaseUISceneGameplay instance with the uiTooltipDialog field assigned.

Advanced Integration:

  • Tooltips are registered automatically via the gameâ€™s database loading pipeline.
  • Dialogs are populated using the Setup() method in UITooltipDialog, which you can override or extend for your UI style.

Dependencies:
This system assumes you are using the BaseUISceneGameplay pattern provided by the MMORPG Kit. You can extend the singleton reference if needed to suit custom scenes.
